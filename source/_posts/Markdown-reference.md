# Markdown 参考

## 概述

Markdown 就是一个轻量的标记语言,适合博客,程序员写文档. HTML有一些关系,

## 现行标准

原始版本比较简化,后来有各种方言,原作者不支持标准化.所以仍然是方言碎片,且有多种写法

* 原始Markdown
* GFM[^1]
* CommonMark[^2]
* pandoc Markdown[^3]

参考链接

## 标准扩展

* 表格
* 代码块, 自动识别代码块,语法高亮

* 公式
* 思维导图
* 目录
* 脚注
* emoji等等
* 定义列表: 词汇表用途
